📖 Cara Menambah Kosakata Baru

  1. Simpan file HTML flashcard baru ke folder yang sama dengan index.html
  2. Buka index.html dengan teks editor (Notepad, VS Code, dll)
  3. Cari bagian const VOCAB_LIST = ` di dalam file

    namafile.html | kata | YYYY-MM-DD | pos | preview | level
  5. Simpan file, refresh browser — kartu baru langsung muncul ✓
